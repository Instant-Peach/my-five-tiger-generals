## 🚨 과대 해석 목록 (우선 확인 필요)

1. 행동력 시스템: "행동력 3" vs "2 액션"

SSOT (rules.md:74):
"한 턴에 행동력 3: [이동] / [공격] / [모의]"

현재 문서 (02-game-rules.md:104):
"각 플레이어는 턴당 2 액션을 수행합니다."

분석:
- SSOT에서는 명확히 "행동력 3"으로 명시
- 현재 문서에서는 "2 액션"으로 임의 변경된 것으로 보임
- PRD(prd.md:44)에서도 "턴 시스템: 한 턴에 기본 2개 행동 수행 (특정 책략으로 +1 가능)"이라고 되어 있어 SSOT 내부 불일치 가능성도 있음

권고: ❗ 원저자 확인 필요 - 행동력이 2인지 3인지 명확히 확정해야 함

---
2. 스탯 시스템: "별(통솔력) = 병력 상한" vs "성(Star) = 방어력"

SSOT (rules.md:34-35):
"⭐ 별 (통솔력): 1~5개 → 별 1개당 병력 2단위(=2만) 까지 통솔 가능."

현재 문서 (02-game-rules.md:80):
"| 성 (Star) | ⭐ | 방어력 / 특수 능력 |"

분석:
- SSOT에서 "별"은 **통솔력(병력 상한)**을 의미
- 현재 문서에서 "성(Star)"은 방어력으로 완전히 다른 의미로 사용
- 이것은 매우 중대한 게임 메커니즘 변경

추가 확인 - SSOT (rules.md:50-54):
해·달 라인 전투
- 공격력 비교: 내 공격력 - 상대 공격력 = 피해

현재 문서 (02-game-rules.md:179):
데미지 = 공격자 공격력 - 방어자 방어력(Star)

권고: 🔴 심각한 불일치 - SSOT에서는 해/달 전투에서 "공격력 vs 공격력" 비교인데, 현재 문서에서는 "공격력 vs 방어력(Star)"으로 변경됨. 원저자 확인 필수

3. 보급 시스템의 과대 해석

SSOT (rules.md:86-89):
- [이동]으로 아군과 포개면 보급을 수행한다.
- 보급을 위해 복귀 대기한 장수는 1턴 후 자동 복귀한다.
- 보급 복귀에는 OUT 카운트가 적용되지 않는다.

현재 문서 (02-game-rules.md:147-149):
- 이동 중 아군 타일(진영 내)을 지나면 보급 발동
- 보급: 병력 일부 회복 (Phase 3에서 상세 구현)

분석:
- SSOT에서는 "아군과 포개면 보급 → 복귀 대기 → 1턴 후 자동 복귀"라는 복잡한 시스템
- 현재 문서에서는 "아군 타일 지나면 보급 → 병력 일부 회복"으로 단순화 해석
- "복귀 대기" 개념과 "OUT 카운트 미적용" 규칙이 완전히 누락

권고: ❗ 원저자 확인 필요 - 보급 시스템이 의도적으로 간소화된 것인지 확인

- 책략 사용을 위한 [모의] 액션은 게임의 핵심 메커니즘 중 하나

권고: ❓ MVP에서 책략이 제외되어 [모의]를 생략한 것으로 보이나, Phase 3에서 책략 구현 시 반드시 추가해야 함

## ❌ 불일치 목록 (수정 필요)

1. 승리 조건: "와해" 누락

SSOT (rules.md:103-105):
1. 노크 세 번
2. 전멸: 상대 병력 전부 소멸
3. 와해: 상대 장수가 모두 완전 퇴장 (일시 OUT은 포함하지 않음)
4. 항복: 상대 항복 선언

현재 문서 (02-game-rules.md:238-243):
| 섬멸 승리 | 상대 모든 기물 OUT | 1 (즉시 승리) |
| 노크 승리 | 노크 3회 달성 | 2 |
| 항복 | 상대가 항복 선언 | - |

분석:
- SSOT에서 "전멸"과 "와해"가 구분되어 있음
- 전멸 = 병력 전부 소멸 (병력 = 0)
- 와해 = 장수가 모두 완전 퇴장 (전사 = OUT)
- 현재 문서에서는 "섬멸 승리"로 통합되어 있음

권고: 전멸과 와해의 차이가 있는지 확인 필요. SSOT에서 병력 시스템과 장수 OUT 시스템이 별도로 존재하는 것으로 보임

2. 패전 처리: 스탯 감소 유무

SSOT (rules.md:92-94):
- 병력 전부 소멸 시 전사 처리되며, 해당 유닛은 인게임에서 완전 소멸한다.
- 스탯 감소 등 패전 페널티는 없다.

PRD (prd.md:59-61):
- 병력 0 = 패전 → 복귀 (능력치 1점 차감)
- 2번째 패전 = 완전 퇴장
- 능력치 차감: 별, 발, 해, 달 중 하나 선택

현재 문서 (02-game-rules.md:83-86):
- 각 기물은 병력 수치를 가짐 (초기값: 기물별 상이)
- 병력이 0이 되면 OUT 상태

분석:
- rules.md(SSOT): "전사 → 완전 소멸, 스탯 감소 없음"
- prd.md: "패전 → 복귀(스탯 1점 감소) → 2번째 패전 = 완전 퇴장"
- 현재 문서: 병력 0 = OUT (상세 미기술)

권고: 🔴 SSOT 내부 불일치 존재 (rules.md vs prd.md). 원저자가 어느 규칙이 최신인지 확인 필요

### 3. 노크 규칙: 별도의 행동으로 노크

SSOT (rules.md:101-102):
- 정의: 상대 진영 끝 칸에 도달한 상태에서만 사용할 수 있는 행동 1개로, 적 진영 자체에 피해를 가하는 행위를 "노크"로 규정한다.
- 주의: 행동을 모두 소진하여 끝 칸에 도달했다면 그 턴에는 노크 불가.

현재 문서 (02-game-rules.md:227-229):
- 상대 **끝 행(Row)**에 기물이 도달하면 노크 1회
- 노크 후 해당 기물은 시작 위치로 복귀

분석:
- SSOT: 노크는 끝 칸 도달 후 별도의 행동으로 수행해야 함 (도달만으로는 노크 아님)
- 현재 문서: 끝 행에 도달하면 자동으로 노크 1회 (도달 = 노크)

권고: 🔴 중요한 규칙 변경 - 노크가 "도달 시 자동"인지 "별도 행동"인지 확인 필요
