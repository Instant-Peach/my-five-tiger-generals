# 오호대장군 (Five Tiger Generals) - 프로젝트 개요

> 작성일: 2026-02-02
> 버전: 1.1.0 (SSOT 정합성 반영)
> 작성자: BMAD Party Mode (Cloud Dragonborn, Samus Shepard, Link Freeman, GLaDOS, Max, Indie)

---

## 1. 프로젝트 비전

**오호대장군**은 삼국지 테마의 1:1 턴제 전략 보드게임입니다.

독특한 **삼각형 테셀레이션 보드**와 **해(Sun)/달(Moon)/전선(Frontline)** 방향성 전투 시스템을 통해 기존 보드게임과 차별화된 전략적 깊이를 제공합니다.

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **전략적 깊이** | 단순한 규칙, 깊은 전략 - "배우기 쉽고 마스터하기 어렵다" |
| **빠른 플레이** | 20-40분 내 한 판 완료 |
| **모바일 우선** | 언제 어디서나 한 판 |
| **공정한 경쟁** | Pay-to-Win 없는 순수 실력 경쟁 |

## 2. 프로젝트 배경

### 2.1 기존 프로젝트 분석

두 개의 기존 프로젝트에서 자산을 계승합니다:

| 프로젝트 | 가져올 것 | 상태 |
|---------|----------|------|
| **Legacy** (`my-five-tiger-generals-legacy/`) | 게임 로직, 기획 문서, 보드 시스템 | 80% 완성 |
| **Main** (`my-five-tiger-generals/`) | 인프라 패턴, SSR, CI/CD | 70% 완성 |

### 2.2 새 프로젝트 목표

- 두 프로젝트의 장점을 결합
- 처음부터 깔끔한 구조로 재구축
- **플레이 가능한 데모 우선** 접근법

## 3. 타겟 사용자

### 주요 타겟

| 세그먼트 | 특성 | 니즈 |
|---------|------|------|
| **전략 게임 매니아** | 체스, 장기, 바둑 경험자 | 새로운 전략적 도전 |
| **삼국지 팬** | 삼국지 IP 선호 | 테마와 몰입감 |
| **캐주얼 모바일 게이머** | 짧은 세션 선호 | 빠른 한 판, 간편한 조작 |

### 플랫폼 우선순위

1. **모바일 웹** (최우선) - PWA로 앱 설치 없이 플레이
2. **데스크톱 웹** - 더 큰 화면에서 전략 분석
3. **네이티브 앱** (향후) - 모바일 스토어 배포

## 4. 핵심 게임 컨셉

### 4.1 삼각형 보드

```
    ▲ ▼ ▲ ▼ ▲
   ▼ ▲ ▼ ▲ ▼ ▲
    ▲ ▼ ▲ ▼ ▲
   ▼ ▲ ▼ ▲ ▼ ▲
    ▲ ▼ ▲ ▼ ▲
   ▼ ▲ ▼ ▲ ▼ ▲
```

- **34 타일**: 메인 30 + 특수 엣지 4
- **Up/Down 방향**: 삼각형 방향에 따른 전술 차이
- **3가지 인접 유형**: 변 접촉, 꼭짓점 접촉, 대각 연결

### 4.2 오호대장군 (五虎大將軍)

각 플레이어는 유비 휘하 5명의 명장을 지휘합니다:

| 장수 | 특성 |
|------|------|
| **관우 (關羽)** | 의리의 무신, 균형 잡힌 핵심 장수 |
| **장비 (張飛)** | 맹장, 높은 공격력 |
| **조운 (趙雲)** | 상산의 용, 높은 기동력 |
| **황충 (黃忠)** | 노익장, 원거리 특화 |
| **마초 (馬超)** | 서량의 맹호, 돌격형 |

### 4.3 방향성 전투

| 공격 방향 | 아이콘 | 설명 |
|----------|--------|------|
| **해(Sun)** | ☀️ | 우측 대각선 공격, Sun 스탯 사용 |
| **달(Moon)** | 🌙 | 좌측 대각선 공격, Moon 스탯 사용 |
| **전선(Frontline)** | ⚔️ | 수평 공격, 교착 상태 유발 |

### 4.4 승리 조건

- **노크 승리**: 상대 끝 구역 도달 후 **별도의 행동으로 노크 3회**
- **전멸 승리**: 상대 병력 전부 소멸
- **와해 승리**: 상대 장수 모두 완전 퇴장 (일시 OUT 제외)
- **항복**: 상대 항복 선언

### 4.5 디자인 스타일

> **레트로 픽셀 아트 삼국지 전술 RPG**

- **스타일**: 삼국지 영걸전 같은 느낌의 **픽셀 아트 스타일**
- **색감**: 레트로 감성의 제한된 팔레트
- **타일 디자인**: 명확한 구분이 가능한 픽셀 기반 타일
- **장수 표현**: 픽셀 아트 스프라이트

> **참고**: 초기에는 "흰 배경 + 검은 먹선 + 캐릭터별 컬러" 스타일을 고려했으나, 현재는 픽셀 아트 방향으로 변경되었습니다.

## 5. 프로젝트 단계

### Phase 1: Local Demo (4주)

> **목표**: 2인 로컬 플레이 가능한 데모

```
Week 1: 보드 렌더링 + 기물 배치
Week 2: 이동 시스템 + 경로 표시
Week 3: 전투 시스템 (해/달/전선)
Week 4: 승리 조건 + 폴리시
```

**완료 기준**:
- [ ] 삼각형 보드에서 2인 번갈아 플레이
- [ ] 해/달/전선 공격 정상 동작
- [ ] 노크 3회 시 승리 판정
- [ ] 모바일 브라우저 동작

**배포**: Cloudflare Pages (정적)

---

### Phase 2: Online Demo (4주)

> **목표**: 1:1 온라인 매칭 플레이

```
Week 5-6: Colyseus 게임 서버 구축
Week 7: 매칭 시스템 + 룸 관리
Week 8: 실시간 동기화 + 재연결
```

**완료 기준**:
- [ ] 랜덤 매칭으로 상대 찾기
- [ ] 실시간 게임 상태 동기화
- [ ] 연결 끊김 시 재연결 지원
- [ ] 비정상 종료 처리

**배포**: + Docker (백엔드)

---

### Phase 3: Full Game (8주+)

> **목표**: 완전한 게임 서비스

```
전술 시스템 (20개 전술)
계정 시스템 (소셜 로그인)
랭킹 시스템 (ELO 레이팅)
리플레이 시스템
튜토리얼
```

**배포**: + Supabase (DB) + 모니터링
